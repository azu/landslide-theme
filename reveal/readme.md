# reveal

landslide theme with [reveal.js](https://github.com/hakimel/reveal.js "reveal.js")

    landslide config.cfg -c

need `--copy-theme` option.

    config.cfg example

    [landslide]
    theme  = path/to/reveal
    LINENOS = no
    copy_theme = True # dones't work...
    relative = False
    embed = True
    destination = index.html
