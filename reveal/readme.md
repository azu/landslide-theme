# reveal

landslide theme with [reveal.js](https://github.com/hakimel/reveal.js "reveal.js")

    > landslice config.cfg -c

need `--copy-theme` option.


    [landslide]
    theme  = path/to/reveal
    LINENOS = no
    copy_theme = True
    relative = False
    embed = True
    destination = index.html
